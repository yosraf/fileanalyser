# fileanalyser
## Run the application
 mvn clean install <br>
 mvn spring-boot:run <br>
#### OR
mvn clean install <br>
mvn package <br>
java -jar target/fileanalyser-0.0.1-SNAPSHOT.jar

## Rest API
`GET curl --location 'http://localhost:8080/mp4-analysis?url=https%3A%2F%2Fdemo.castlabs.com%2Ftmp%2Ftext0.mp4'`'
